This project allows you to create any type of safetiness password that includes lowercase letters, uppercase letters, symbols and numbers.
At first you enter minimum length of password, then you enter maximum length of password. In case you min. length is bigger than max. length, the code will inform you about the mistake and will ask you to start from the begining.
After that you will be asked if you want to include lowercase or uppercase letters, then you will be asked if you want any symbols or numbers, again if you denied every given choices you will be informed about the error and will be asked to start creating new password from the begining.
In the end you will be given your generated password with it's quality grade from weak to very strong.
